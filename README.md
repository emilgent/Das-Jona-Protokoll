# Das Jona-Protokoll: Ein interaktives Büro-Dilemma

Willkommen beim **Jona-Protokoll** – einem textbasierten Adventure, das stark von modernen Meta-Spielen wie *The Stanley Parable* inspiriert ist.

Du schlüpfst in die Rolle von Jona, einem ganz normalen Angestellten an einem ganz normalen Schreibtisch. Doch heute ist etwas anders: Das gesamte Büro ist menschenleer. Und da ist diese Stimme in deinem Kopf – der Erzähler –, der ganz genaue Vorstellungen davon hat, was du als Nächstes tun solltest. Wirst du seinen Anweisungen folgen? Oder wirst du versuchen, das System zu brechen?

---

## 🎮 Features

* **Tiefgehende Narrative:** Deine Entscheidungen verändern den Verlauf der Geschichte spürbar.
* **7 komplett unterschiedliche Enden:** Vom stinknormalen Feierabend über das totale System-Paradoxon bis hin zum Ausbruch aus der simulierten Realität.
* **Persistente Spielwelt:** Das Spiel besitzt ein Gedächtnis! Es speichert deine Durchgänge im Hintergrund und erinnert sich daran, wie oft du es schon versucht hast.
* **Ausgiebige Erkundung:** Jeder Raum bietet versteckte Gegenstände und mehrere Interaktionsmöglichkeiten.
* **Geheime Easter Eggs:** Für besonders neugierige Spieler gibt es ungelistete Optionen, um die Logik des Spiels komplett auszuhebeln.

## 🚀 Installation & Start

Um das Spiel zu spielen, benötigst du lediglich eine aktuelle Installation von **Python 3**.

1. **Code herunterladen:** Kopiere den Python-Code in eine Datei und benenne sie `main.py`.
2. **Terminal öffnen:** Navigiere in der Konsole/im Terminal in den Ordner, in dem die Datei liegt.
3. **Spiel starten:** Gib den folgenden Befehl ein:
```bash
python main.py

```



---

## 🛠️ Spielanleitung

* **Eingabe:** Das Spiel präsentiert dir in jedem Raum verschiedene nummerierte Optionen (z. B. `[1]`, `[2]`). Tippe einfach die gewünschte Zahl ein und drücke `Enter`.
* **Geduld:** Der Erzähler spricht in Echtzeit. Die Texte werden wie auf einer alten Schreibmaschine eingeblendet, um die Atmosphäre zu verdichten.
* **Das Unmögliche versuchen:** Manchmal lohnt es sich, Eingaben auszuprobieren, die *nicht* auf dem Bildschirm stehen. Wer weiß, was passiert, wenn man dem System die richtigen Admin-Befehle füttert...

## 📂 System-Dateien

Das Spiel erstellt beim ersten Start automatisch eine kleine Datei im selben Verzeichnis:

* `save.txt`: Speichert die Anzahl deiner bisherigen Spielrunden, um den Dialog des Erzählers dynamisch anzupassen. Lösche diese Datei, wenn du deinen Fortschritt komplett zurücksetzen willst.
